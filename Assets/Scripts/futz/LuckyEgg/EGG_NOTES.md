# Initial Setup

do this once PER PROJECT

- add this to manifest.json:

```
"com.luckyegg.egg": "file:../../EGG/Assets/LuckyEgg",
```

# User Setup

do this once PER USER/workspace

- get EGG repo and put it in the same dir as project folder
- e.g. `C:/MyProject/` & `C:/EGG/`
